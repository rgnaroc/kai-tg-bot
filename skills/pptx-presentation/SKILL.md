# Skill: Создание PPTX презентаций

## Когда использовать
- Пользователь говорит: "сделай презентацию", "накидай слайды", "оформи в pptx"
- Нужно визуализировать структурированные данные
- Подготовка доклада, лекции, отчёта

## Как использовать

### 1. Подготовь JSON-описание презентации
Передай скрипту JSON через stdin или файл.

### 2. Запусти генерацию
```bash
python3 /root/skills/pptx-presentation/pptx_builder.py < input.json
# или
python3 /root/skills/pptx-presentation/pptx_builder.py --output presentation.pptx < input.json
```

### 3. Открой результат
Файл сохраняется в текущей директории. Открыть командой:
```bash
open output.pptx  # или send пользователю
```

## Формат входного JSON

```json
{
  "title": "Заголовок презентации",
  "subtitle": "Подзаголовок",
  "author": "Автор",
  "theme": {
    "primary_color": "1B3A5C",
    "accent_color": "2196F3",
    "bg_color": "FFFFFF",
    "font_name": "Roboto"
  },
  "slides": [
    {"type": "title"},
    {"type": "bullets", "title": "Раздел", "items": ["Пункт 1", "Пункт 2"]},
    {"type": "table", "title": "Данные", "headers": ["A","B"], "rows": [["1","2"]]},
    {"type": "image_text", "title": "Слайд", "text": "Описание", "image_path": ""},
    {"type": "process", "title": "Процесс", "steps": ["Шаг 1", "Шаг 2"]},
    {"type": "code", "title": "Код", "code": "print('hello')", "language": "python"},
    {"type": "quote", "text": "Цитата", "source": "Автор"}
  ]
}
```

## Типы слайдов
| Тип | Описание |
|-----|----------|
| `title` | Титульный слайд с заголовком + подзаголовком |
| `bullets` | Заголовок + список буллитов |
| `table` | Таблица с заголовками и строками |
| `image_text` | Текст слева, изображение справа (или наоборот) |
| `process` | Заголовок + нумерованные шаги процесса |
| `code` | Блок кода с подсветкой синтаксиса |
| `quote` | Крупная цитата с указанием источника |
| `section` | Слайд-разделитель (только заголовок) |

## Декоративные фоны
По умолчанию на каждом слайде (кроме титульного и секций) рисуются
полупрозрачные геометрические фигуры: круги, треугольники.
Это делает презентацию визуально богаче без загрузки картинок.
Можно отключить, добавив в тему:
```json
"theme": {
  "background_style": "plain"
}
```

## Палитра по умолчанию
- primary: #1B3A5C (тёмно-синий)
- accent: #2196F3 (голубой)
- bg: #FFFFFF (белый)
- text: #333333 (тёмно-серый)
